# Task 01 - Use GitHub Copilot to create an Issue

## Description

In this task you will create a new issue that will describe the requirements to update the solution from .NET 6 to the latest LTS version of .NET.

You can create issues using GitHub Copilot directly in GitHub online, or from Visual Studio Code. In an earlier exercise you used GitHub online. For this exercise you will use Visual Studio Code.

## Success Criteria

- You have created a new issue using GitHub Copilot in Visual Studio Code.

## Learning Resources

- [Using GitHub Copilot to create or update issues](https://docs.github.com/en/copilot/how-tos/use-copilot-for-common-tasks/use-copilot-to-create-or-update-issues)

## Key Tasks

### 01: Open Copilot Chat

From Visual Studio Code, launch Copilot Chat:

![Opening Copilot Chat from VS Code](../../media/0501_vsc_openchat.png)

### 02: Set the Copilot Chat context

Set the context to the /src directory. You can click the attachment icon in the chat window and select from the dropdown list Files & Folders, then `src`.

![Set the Copilot Chat context](../../media/0501_set_chat_context.png)

### 03: Create a GitHub issue with Copilot Chat

In the chat window type `/create-issue ` followed by your prompt text. In this case you want to prompt indicating you want to update the solution to the LTS version of .NET.

```text
Update .NET to the current LTS version
```
![VS Code Copilot Chat](../../media/0501_code_copilot_chat_input.png)

The result will be something similar to this:

![VS Code Copilot Chat](../../media/0501_code_copilot_chat_output.png)

Once you are satisfied with the issue description, press allow to create the issue. If all goes as expected, a success message will be shown:

![Create Issue Success](../../media/0501_code_copilot_chat_issue_created.png)

Followed by the recommended next steps:

![Create Issue Next Steps](../../media/0501_code_copilot_chat_issue_next%20steps.png)

## Summary

You've completed this task. You have created a new issue using GitHub Copilot in Visual Studio Code.
